# Gözde Akın — Profesyonel Fotoğraflar

## Hızlı kurulum (önerilen)

1. Memorial önlüklü profesyonel portrenizi şuraya kaydedin:

   `gozde-portrait-source.jpg`

2. Terminalde çalıştırın:

   ```bash
   npm run portrait:install
   ```

Bu komut aşağıdaki dosyaları otomatik oluşturur:

| Dosya | Boyut | Kullanım |
|-------|-------|----------|
| `gozde-memorial-hero.jpg` | 1200×1500 | Ana sayfa hero, hakkımda, iletişim |
| `gozde-office-portrait.jpg` | 1200×1500 | Yedek / ofis portresi |
| `profile.jpg` | 1200×630 | SEO, Open Graph |

## Manuel değiştirme

Kaynak fotoğrafı doğrudan `gozde-memorial-hero.jpg` olarak da kaydedebilirsiniz.

## Reel ve başarı görselleri

| Dosya | Açıklama |
|-------|----------|
| `reel-5-basit-kural.jpg` | 5 Basit Kural reel kapağı |
| `reel-karniyarik-otu.jpg` | Karnıyarık Otu reel kapağı |
| `reel-tatli-krizi.jpg` | Tatlı Krizi reel kapağı |
| `reel-insulin-direnci.jpg` | İnsülin Direnci reel kapağı |
| `success-ekip-isi.jpg` | Başarı hikayesi grafiği |

**Reel boyutu:** 1080×1920 (9:16)  
**Format:** JPG veya WebP, optimize edilmiş (< 400 KB)
